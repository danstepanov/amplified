Custom.js

All the javascript is in the custom.js file. USe it to modify settings for Sliders, Portfolio, Twitter, Flickr Widget, etc.

//ISOTOPE

$(document).ready(function() {

      // cache container
var $container = $('#container-portfolio');
// initialize isotope
$container.isotope({
  // options...
    
});

// filter items when filter link is clicked
$('#filters a').click(function(){
  var selector = $(this).attr('data-filter');
  $container.isotope({ filter: selector });
  return false;
});
        
      
});

  $(function($){
"use strict"; 
  $("#twitterfeed").tweet({
    username: "bypixels", /*Enter your Twitter username */
    modpath: 'js/twitter/', /* director of the twitter folder : ABSOLUTLY DIRECT LINK OF YOUR TWITTER FOLDER */
    page: 1, /*How many tweets to show at a time*/
    avatar_size: null, 
    count: 5, /*Total counts of tweets. */
    loading_text: "Loading latest tweets..." /*Loading text*/
  });

Retina Icons

The icon itself is powered by font awesome, the code is this:

<i class="fa_icon icon-align-justify icon-3x"></i>

How to change an icon?

As you can see icon-align-justify is the name of the icon, you have to change the name to change the icon. Here you can see a complete list of icons, select the name and replace it.

The complete Font Awesome 3.2.1 icon reference


Size of the icon

<i class="fa_icon icon-align-justify icon-3x"></i>

Has you can see the icon have a icon-3x. To change the size of an icon just change the number like

icon-2x icon-3x icon-4x


Services icons using Retina Icons

Here is the structure of the services icons

<div class="services_icon_container hi-icon-effect-1 hi-icon-effect-1a">
  <div class="span6">
    <div class="icon_wrapper"><a href="#" class="hi-icon"><i class="fa_icon icon-align-justify icon-3x"></i></a></div>
    <h2 class="uppercase">100% Responsive</h2>      
  </div><!--span6-->
</div><!--services_icon_container-->

There is a container call .services_icon_container what wraps all the icons. To create a icon do this 

<div class="icon_wrapper"><a href="#" class="hi-icon"><i class="fa_icon icon-align-justify icon-3x"></i></a></div>

Video Style Background

To add a video just go to youtube, select your video. Click in share, and copy the URL. Then go to your index-4.html file where you have the video and paste the URL from youtube embed code. (look at screenshot two). Click in the image to amplify it.
